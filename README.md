# eurocard_mux

![HEADER](./PICTURES/eurocard_mux_iso.png) <!-- 3D rendered pretty view -->

10ch -> 10A - 10B mux board for ert measurements. Uses dpdt latching relays to switch between A and B channels. and has front and rear connectors for easy wiring. Allso includes diodes for indicating relay state.
